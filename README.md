# Izzziland Website

## Описание

Izzziland Website — это официальный веб-сайт Minecraft-сервера Izzziland. Проект предназначен для предоставления игрокам актуальной информации о сервере, его правилах, игровых сезонах и дополнительных ресурсах. Сайт обеспечивает удобную навигацию и быстрый доступ к основным функциям проекта.

## Функционал

* Главная страница с информацией о сервере;
* Просмотр правил сервера;
* Раздел с описанием игровых сезонов;
* Вики-раздел с полезной информацией;
* Просмотр карты игрового мира;
* Отображение текущего онлайна сервера;
* Копирование IP-адреса сервера в буфер обмена;
* Переходы на официальные социальные сети проекта;
* Адаптивный интерфейс для различных устройств.

## Технологии

* HTML5
* CSS3
* JavaScript (ES6)
* API mcsrvstat.us
* GitHub

## Установка и запуск

1. Склонировать репозиторий:

```bash
git clone https://github.com/USERNAME/REPOSITORY.git
```

2. Перейти в папку проекта:

```bash
cd REPOSITORY
```

3. Открыть файл `main.html` в браузере.

Либо использовать расширение Live Server в Visual Studio Code для локального запуска проекта.
